# Components

Chart components are lightweight Python objects consumed by a chart container.
They do not render independently.

- [Annotations](/docs/xy/components/annotations/) add rules, bands, labels, markers,
  arrows, thresholds, and callouts.
- [Chart chrome](/docs/xy/components/chart-chrome/) configures legends, tooltips,
  colorbars, modebars, themes, and interaction behavior.
- [Axes and scales](/docs/xy/core-concepts/axes-and-scales/) covers `x_axis` and `y_axis`.

For a compact inventory of every factory, see the
[API reference](/docs/xy/api-reference/components/).
